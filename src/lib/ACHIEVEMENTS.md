# Achievements Documentation

This document describes all achievements available in the Cube app, their requirements, and how to unlock them.

## Smart Cube Requirement

Achievements marked with ðŸ”Œ **require a smart cube** to track properly. Manual timer solves do not count toward these achievements.

## Categories

- **Grind** - Cumulative achievements based on total solves/moves
- **Smart Cube** - Achievements requiring specific move patterns or timing
- **CFOP** - Achievements related to CFOP method stages and skips
- **Anomaly** - Unusual or rare occurrences
- **Streak** - Daily consistency achievements

| Achievement | Tracking Key |
|-------------|--------------|
| The Centurion / Millennial / Ten Thousand / Cube Voyager | `totalSolves` |
| Finger Gymnastics | `totalMoves` |
| Marathon Hand | `totalRotationDegrees` |
| The Sniper | `crossUnder8Moves` |
| F2L Flow | `f2lNoPause` |
| PLL Powerhouse | `pllUnder4s` / `pllUnder3s` / `pllUnder2s` / `pllUnder1_5s` / `pllUnder1s` |
| TPS Beast | `tpsOver5Solves` |
| OLL Skip Hunter | `ollSkips` |
| PLL Skip Hunter | `pllSkips` |
| God's Number | `godsNumberSolves` |
| Full Step Master | `fullStepSub15` |
| The Glitch | `perfectMoveMatches` |
| Inefficient Genius | `sub20With80Moves` |
| Streak Starter | `currentStreak` |
